# AI Playground

## Español

Este repositorio es mi laboratorio personal de aprendizaje de inteligencia artificial. Mientras que [AI Workflows](https://github.com/alexvinola/AI_Workflows) documenta la parte aplicada — sistemas, agentes y arquitecturas listas para producción — este espacio es diferente: aquí experimento, me equivoco y aprendo desde cero los fundamentos que hay detrás.

El foco está en entender cómo funcionan los modelos por dentro: cómo se cargan, cómo se ejecutan en local, cómo se ajustan a un dominio concreto y cómo se evalúan. Todo con código real, no teoría.

### Estructura

El repo está organizado por áreas de aprendizaje. Cada carpeta contiene notebooks de Jupyter con el código y las explicaciones integradas.

| Carpeta | Contenido |
|---------|-----------|
| [01_ollama](./notebooks/01_ollama/) | LLMs en local: llamadas, system prompts, temperatura, memoria |
| [02_huggingface](./notebooks/02_huggingface/) | Transformers, inferencia directa, modelos open source |
| [03_finetuning](./notebooks/03_finetuning/) | Ajuste de modelos con LoRA/QLoRA sobre datos propios |
| [04_evaluacion](./notebooks/04_evaluacion/) | Métricas, benchmarks y comparación de modelos |

> El repo se irá actualizando a medida que avance en el aprendizaje.

## English

This repository is my personal AI learning lab. While [AI Workflows](https://github.com/alexvinola/AI_Workflows) documents the applied side — production-ready systems, agents and architectures — this space is different: here I experiment, make mistakes and learn the underlying fundamentals from scratch.

The focus is on understanding how models work under the hood: how they are loaded, how they run locally, how they are fine-tuned to a specific domain and how they are evaluated. All with real code, not theory.

### Structure

The repo is organized by learning areas. Each folder contains Jupyter notebooks with code and explanations integrated together.

| Folder | Content |
|--------|---------|
| [01_ollama](./notebooks/01_ollama/) | Local LLMs: calls, system prompts, temperature, memory |
| [02_huggingface](./notebooks/02_huggingface/) | Transformers, direct inference, open source models |
| [03_finetuning](./notebooks/03_finetuning/) | Model fine-tuning with LoRA/QLoRA on custom data |
| [04_evaluacion](./notebooks/04_evaluacion/) | Metrics, benchmarks and model comparison |

> The repo will be updated progressively as I advance in my learning.
